# 🎓 Student Management System

## 📌 Description

This is a **console-based application** developed in Python that allows managing student information.

The system supports basic CRUD operations:

* Create
* Read
* Update
* Delete

It also includes **data persistence using JSON files**, input validation, and error handling.

---

## 🌎 Descripción (Español)

Esta es una aplicación de consola desarrollada en Python que permite gestionar información de estudiantes.

El sistema permite realizar operaciones CRUD:

* Crear
* Consultar
* Actualizar
* Eliminar

Incluye persistencia de datos en archivos JSON y validaciones.

---

## 🚀 How to Run / Cómo ejecutar

### 🔹 Requirements

* Python 3.x installed

### 🔹 Steps

```bash
# Clone the repository (optional)
git clone <repository-url>

# Enter project folder
cd student_system

# Run the program
python main.py
```
